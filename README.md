## Execute no seu computador

1 - Backend
* Start Backend:
  * cd backend (Entrar na pasta Backend);
  * Run: npm i (Instalar dependências);
  * Run: npm start (Iniciar o Backend).

2 - Frontend
* Start Frontend:
  * cd frontend;
  * Run: npm i (Instalar dependências);
  * Run: npm start (Iniciar o Frontend).
